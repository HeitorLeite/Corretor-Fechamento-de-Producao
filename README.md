# Corretor-Fechamento-de-Producao

Esse programa foi feito para ajudar na produtividade do meu trabalho, ele ajuda a coletar dados de uma planilha para colocar dentro do sistema da empresa.

Esse programa é um conversor de dados do Excel para formato CSV ele lê uma planilha de Fechamento da Produção dos Prestadores e transforma em arquivo CSV no formato esperado.

### Funcionamento 

1 - Com a sua planilha pronta coloque o arquivo na pasta "Planilhas".
<br>
2 - Após isso baixe o programa e a dependencia necessária "pip install openpyxl".
<br>
3 - O programa então vai pegar os dados da planilha e transformar no seguinte padrão dentro do arquivo .csv.
<br>
3.1 - 1202;1202;;1127.01S ou 230765;230765;;2754.46.
<br>
4 - Finalizando o arquivo final corrigido será salvo na pasta Saida.
